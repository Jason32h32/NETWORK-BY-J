# Introduction

Modern organizations rely on computer networks to enable communication, resource sharing, and business operations. Every device including computers, servers, printers, and wireless devices must communicate efficiently and securely within the network.

This project demonstrates how a small enterprise network can be designed and implemented using Cisco Packet Tracer while following networking concepts covered in the Cisco CCNA 200-301 curriculum.

---

# What is Computer Networking?

Computer networking is the practice of connecting multiple devices to allow them to communicate and exchange data.

A well-designed network enables organizations to:

- Share files and resources
- Access centralized services
- Connect to internal and external networks
- Improve communication
- Increase operational efficiency

Networking is one of the fundamental building blocks of Information Technology and Cybersecurity.

---

# What is Cisco CCNA 200-301?

Cisco Certified Network Associate (CCNA) 200-301 is Cisco's associate-level networking certification.

It provides foundational knowledge in networking technologies, including:

- Network Fundamentals
- IPv4 & IPv6 Addressing
- Ethernet Switching
- Routing
- Network Services
- Wireless Networking
- Network Security
- Automation Fundamentals

The concepts implemented in this project align with the practical skills expected from an entry-level network engineer.
